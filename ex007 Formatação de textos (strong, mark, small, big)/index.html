Aprendendo as formatação de texto

Negrito/Destaque

Para utilizar o negrito use a tag B (tag não semântica)

Ela ficaria assim Não utilizar tag b, pois ela não é semântica.

A tag semântica de negrito é Strong, mas deixe para mexer no css.

Italico/Ênfase

tag em italico utiliza a tag I (não semântica)

A tag semântica para italico é Em

Essa pode usar pois é semântica, Mas como diz acima, é melhor utilizar as css!

Marcação no texto

Para fazer uma marcação no texto utiliza a tag Mark

Assim você pode marcar as linhas e pode alterar as cores usando css, também pode usar a ccs dentro tag Mark.

Textos Grandes e Pequenos


Para textos Grandes usa a tag big(modelo que esta obsoleto)

Para textos Pequenos utiliza a tag small.

Textos deletados

Utiliza a tag del para mostrar que o conteúdo esta excluído, você poderá ver e ler, mas mostrará um risco.

exemplo: Eis um texto excluído..

Texto inserido

ultiliza-se a tag <ins> para inserir, ficará sublinhado.

exemplo: Este texto está sublinhado

Texto sobrescrito

Para fazer um texto sobrescrito utiliza a tag <sup>.

Exemplo: x20+4.

Texto subscrito

Utiliza-se a tag <sub>.

exemplo: H 2O.
